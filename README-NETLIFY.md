# 🚚 TRANSPORTE WILLMORE - Frontend

Sistema de rastreo de paquetes en tiempo real para República Dominicana.

## 🌐 Deployment en Netlify

Este proyecto está configurado para desplegarse automáticamente en Netlify.

### 📋 Requisitos Previos
- Cuenta activa en [Netlify](https://netlify.com)
- Archivo `frontend-netlify.zip` (ya creado)

### 🚀 Pasos para Desplegar

1. **Ve a Netlify:**
   - Accede a [netlify.com](https://netlify.com)
   - Inicia sesión en tu cuenta

2. **Crear Nuevo Sitio:**
   - Click en "Add new site"
   - Selecciona "Deploy manually"

3. **Subir el Archivo:**
   - Arrastra y suelta el archivo `frontend-netlify.zip`
   - O haz click para seleccionar el archivo

4. **Configuración Automática:**
   - Netlify detectará automáticamente la configuración
   - Comando de build: `npm run build`
   - Directorio de publicación: `build/`

5. **¡Listo!**
   - Tu sitio estará online en segundos
   - Netlify te dará una URL única (ej: `https://amazing-site.netlify.app`)

## ✨ Características

- 🗺️ **Mapa Interactivo**: Rastreo en tiempo real con Leaflet
- 👨‍💼 **Panel de Administración**: Gestión completa de paquetes y conductores
- 💼 **Solicitud de Trabajo**: Sistema de registro para conductores
- 📱 **Responsive Design**: Optimizado para móviles y desktop
- 🔄 **WebSockets**: Actualizaciones en tiempo real
- 💳 **Sistema de Pagos**: Integración con pasarelas de pago

## 🛠️ Tecnologías

- **React 18** - Framework frontend
- **Leaflet** - Mapas interactivos
- **Socket.io** - Comunicación en tiempo real
- **Tailwind CSS** - Estilos modernos
- **React Router** - Navegación SPA
